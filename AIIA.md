# AI範圍
AI(推薦系統 +機器學習 +群集智慧) > ML(機器學習) > ANN(類神經網路) > 深度學習(生成式AI)

# AI中的環境類組
完全可觀察(Fully observable) vs. 部分可觀察(Partially observable)

單元劇式(Episodic) vs. 連續劇式(Sequential)

確定性(Deterministic) vs. 非確定性/隨機性(Stochastic)

靜態(Static) vs. 動態(Dynamic)

離散(Discrete) vs. 連續(Continuous)

單代理人(Single agents) vs. 多代理人(Multiple agents)

# IA的完整結構
Environment(環境), Sensors(感測器), Actuators(執行器),

Decision-making mechanism(推論/決策機制), Learning system(學習系統/改良機制).

# IA的四大特性(PEAS)
成果/表現(P), 環境(E), 執行工具/執行器(A), 感測器(S).

# IA的類型
簡易的反射式代理人=>預先定義好的規則,

基於模型的反射式代理人=>環境狀態的模型,

基於目標的反射式代理人=>目標,

基於實用效益的代理人=>能達成目標的行動方案不只一個時，靠效用函式來評估可行的效益,

學習型代理人(強化學習)=>能與時俱進、並跟上環境變化的學習機制.

# 學習型代理人的基本元件
學習模組, 評判與回饋機制, 績效評估模組, 行動/改善方案產生器.

# 環境特質類組分析(ex.)
下西洋棋(不限時)=>完全可觀察、非單元劇式、非確定性、靜態、離散、多代理人,

下西洋棋(限時)=>完全可觀察、非單元劇式、非確定性、靜態(1/2)、離散、多代理人,

取物機械手臂(人)(固定不動的)=>完全可觀察、單元劇式、確定性、靜態、非離散、非多代理人,

取物機械手臂(人)(非固定不動)=>非完全可觀察、非單元劇式、非確定性、非靜態、非離散、多代理人,

自駕車(輔助駕駛)=>非完全可觀察、非單元劇式、非確定性、非靜態、非離散、多代理人,

醫療診斷(AI專家系統)=>非完全可觀察、非單元劇式、非確定性、非靜態、非離散、多代理人(1/2),

AI自動點小魚/蝦苗=>完全可觀察、單元劇式、確定性、靜態(1/2)、離散、多代理人(1/2).

# AI自動點小魚/蝦苗的PEAS
P=>準備客戶意見回饋表，訪問客戶系統是否正確計算數量，根據客戶回答的真實性進行加權分數計算，如果客戶隨便回答則加權分數不高，反之則加權分數很高。

E=>準備一個裝水透明的盒子，底部環境使用白色以使環境單純化；如果需要光源則需要從盒子底部打上來以避免影子，使環境完全可觀察。

A=>從感測器獲得的靜態圖片後，使用行動裝置該執行器和內部寫的機器學習演算法程式來做小魚/小蝦苗自動點算系統。

S=>拍攝小魚/小蝦苗所使用之行動裝置的鏡頭。

# 依運作方式區分
AI的chatbot, 基於規則的chatbot, 綜合型的chatbot.

# 關鍵詞, 實體, 意圖
自然語言處理/NLP = 自然語言理解/NLU(意圖分類|語意辨識) + 自然語言生成/NLG

# Chatbot之評量指標
客戶滿意度, 完成度, 退出率|再使用率.

# 推薦系統之主要類型
協同過濾, 內容過濾, 混合推薦系統.

# 推薦系統的缺點及限制
冷啟動(用戶, 物品, 系統), 長尾問題(資料稀疏)

# AI模型
監督式學習(分類, K-NN演算法), 非監督式學習(分群).

# 數值/量化空間的距離
幾何距離, 曼哈頓距離, 漢明距離.

# LLMS
文本資料蒐集, 模型架構及參數設定, 模型學習/訓練->通用式LLM, 模型微調(強化->專業化、客製化).

# LLM模型訓練
預訓練, 監督式模型微調, 兩個階段強化式學習.
